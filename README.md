# Trackify 🚀

A full-stack project management dashboard (Mini CRM) built for teams to manage projects and tasks efficiently.

## Tech Stack

**Frontend**
- React (Vite) + JavaScript
- Tailwind CSS
- Zustand (state management)
- React Query (server state)
- Axios
- React Router DOM

**Backend**
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- Bcryptjs
- Express Validator

## Features

- 🔐 JWT Authentication (Signup/Login)
- 📋 Project & Task Management (CRUD)
- 👥 Assign tasks to users
- 📊 Status tracking (Todo / In Progress / Done)
- 🔍 Search & Filter tasks
- 📄 Pagination
- 🛡️ Role-based access (Admin/User)
- 🌑 Dark theme (Linear.app inspired)
